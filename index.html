Proyecto fin de Ciclo formativo grado superior DAM

Este proyecto final de ciclo me permitió poner en práctica muchos de los conocimientos adquiridos durante mis estudios!

Esta app hace uso de una base de datos local SQLite y de la API REST de Aemet para la obtención de los datos mediante peticiones HTTP
